# FreshTrace 🍽️

FreshTrace is a food transparency and traceability platform that helps users understand exactly where their food comes from before ordering.

## Features

- Restaurant transparency scores
- Ingredient sourcing information
- Freshness tracking
- Dish details and nutritional information
- Customer support portal
- Business dashboard
- Cart management

## Technology Stack

- Python
- Streamlit
- GitHub
- Streamlit Community Cloud

## Live Demo

https://freshtrace-txdzmg8abiwgztxt7rnddc.streamlit.app

## Vision

FreshTrace promotes trust, transparency, and informed food choices by making supply chain information accessible to customers.
